# FEXORA-059: Snippets & Quick-Actions

**Phase:** 2 — Sprint 3
**Prioritat:** Mittel
**App:** Moderator
**Aufwand:** 8-12h
**Status:** Open

---

## Beschreibung

Text-Vorlagen, Medien-Bibliothek und Preis-Buttons fur effizientes Moderator-Arbeiten.

## Aufgaben

- [ ] **Text-Vorlagen (Snippets):**
  - Vordefinierte Schnellantworten (pro Creator konfigurierbar)
  - Snippet-Bibliothek: Kategorien (Begruessen, Upsell, FAQ, Verabschiedung)
  - Einfugen per Klick oder Shortcut
  - Variable: {{username}} wird durch User-Name ersetzt
- [ ] **Medien-Bibliothek:**
  - Zuletzt verwendete Medien (Quick Access)
  - Creator-spezifische Medien (Teaser-Bilder, Preview-Clips)
  - Drag & Drop in Chat
- [ ] **Preis-Buttons (Upsell):**
  - Quick-Buttons fur Content-Empfehlungen
  - Content aus Creator-Portfolio wahlen
  - "Empfehlung senden" -> Content-Link + Preview im Chat
  - PPV-Message erstellen (Preis setzen + senden)

## Akzeptanzkriterien

- Snippets konnen per Klick eingefugt werden
- Medien-Bibliothek zeigt relevante Medien
- Preis-Buttons senden Content-Empfehlungen
- Snippets pro Creator anpassbar
